# 💤 Sleep Disorder Prediction App
Aplikasi Machine Learning berbasis web untuk memprediksi risiko gangguan tidur berdasarkan data kesehatan dan gaya hidup pengguna.

## 🎯 Tujuan Project
Membangun model Machine Learning untuk klasifikasi gangguan tidur menggunakan beberapa algoritma dan membandingkan performanya.

## 🧠 Algoritma yang Digunakan
- K-Nearest Neighbor (KNN)
- Decision Tree ⭐ (Model Terbaik)
- Random Forest

## 📊 Hasil Model Terbaik (Decision Tree)
- Accuracy  : 97.37%
- F1-Score  : 0.9582
- AUC-ROC   : 1.0000

## ⚙️ Tahapan Machine Learning
1. Data Understanding  
2. Data Preprocessing  
3. Exploratory Data Analysis (EDA)  
4. Data Splitting (70:30, 80:20, 90:10)  
5. Model Building  
6. Hyperparameter Tuning (GridSearchCV)  
7. Model Evaluation  
8. Model Comparison
